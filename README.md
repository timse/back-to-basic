
# Back to Basic

Converts (most) european special characters to basic latin characters

# Which characters does it include?

Currently characters are converted as follows:

```
àáâãå => a
äæ => ae
ç => c
èéêë => e
ƒ => f
ìíîïı => i
ñ => n
òóôõø => o
öœ => oe
ß => ss
š => s
þð => th
ùúû => u
ü => ue
ýÿ => y
ž => z
```
